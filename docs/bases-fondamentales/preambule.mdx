# Préambule

## La posture avant la compétence

:::tip Fondamentaux humains
Avant toute ligne de code ou tout concept technique, il est essentiel d’adopter une **posture d’humilité et de transmission**.<br />
L’informatique, comme tout domaine, s’apprend **par étapes**, **avec patience**, et surtout **avec bienveillance**.

> Respecter un débutant, c’est **valoriser le chemin qu’il entreprend**, il ne faut pas le comparer à un niveau avancé.<br /> **C’est très important aussi de reconnaître que chaque expert a été débutant un jour**.

:::

Ainsi donc il faut :

-   [x] Encourager
-   [x] Expliquer
-   [x] Reformuler si besoin
-   [x] Guider sans mépriser

Voici des compétences aussi fondamentales que **Bash**, **Git** ou **la cryptographie**.<br />
**Apprendre à apprendre**, **apprendre à accompagner**, c'est la clé qui fera de vous un professionnel complet.
